Database KamarOTO Structure
=================================================
This table includes the 'users', 'co_profiles', and 'mitra_profiles' structures after the revision, where the 'users' table is used only for initial registration (pending approval), and The approved data will be moved to the respective profile table.

-------------------------------------------------
TABEL: users (for all registrations data relation)
-------------------------------------------------
id                INT (PK, Auto Increment)        - user ID unique
name              VARCHAR(100)                    - User Fullname
email             VARCHAR(100) UNIQUE             - Email login
password          VARCHAR(255)                    - Encyption Password
role              ENUM('admin','mitra','co','customer') - User Type
status            ENUM('pending','approved','rejected','active') - Account Status
phone             VARCHAR(20)                     - Phone/Whatsapp number
created_at        DATETIME                         - Time Registration

-------------------------------------------------
TABEL: co_profiles (profil Captain Officer)
-------------------------------------------------
id                INT (PK, Auto Increment)        - ID profile CO ( Captain Officer )
user_id           INT (FK → users.id)             - User Reference
name              VARCHAR(100)                    - User Fullname
password          VARCHAR(255)                    - Encyption Password
birth_date        DATE                            - Birth Date
gender            ENUM('male','female')           - Gender
address_province  VARCHAR(100)                    - Province of domicile
address_city      VARCHAR(100)                    - City/Regency
address_subdistrict VARCHAR(100)                  - District
address_village   VARCHAR(100)                    - Village
address_detail    TEXT                            - Address detailed
job               VARCHAR(100)                    - Job type
marital_status    VARCHAR(50)                     - Marriage Status
education         VARCHAR(50)                     - Last education
selfie_url        VARCHAR(255)                    - Picture images

-------------------------------------------------
TABEL: mitra_profiles (profil Mitra)
-------------------------------------------------
id                        INT (PK, Auto Increment) - Mitra Profile ID
user_id                   INT (FK → users.id)      - User Reference
password                  VARCHAR(255)             - Encyption Password
pic_name                  VARCHAR(100)             - PIC Name (Person In Charge)
pic_phone                 VARCHAR(20)              - Phone/Whatsapp PIC number
pic_email                 VARCHAR(100)             - Email PIC
pic_status                ENUM('pemilik','pengelola') - Jabatan/status PIC
owner_name                VARCHAR(100)             - Business owner name
owner_phone               VARCHAR(20)              - Phone/Whatsapp owner number
owner_email               VARCHAR(100)             - Owner mail
owner_ktp                 VARCHAR(50)              - KTP owner number
owner_address_province    VARCHAR(100)             - Province address owner
owner_address_city        VARCHAR(100)             - City/Regency address owner
owner_address_subdistrict VARCHAR(100)             - District address owner
owner_address_village     VARCHAR(100)             - Village address owner
owner_address_detail      TEXT                     - Detail address owner
business_type             ENUM('jual_beli_kendaraan','bengkel','cuci_kendaraan','jual_beli_sparepart','sewa_kendaraan','insurance_consultant','pembiayaan','biro_jasa') - Bussiness type
business_entity           VARCHAR(50)              - Types of business entities (CV/PT/dll)
business_address_province VARCHAR(100)             - Bussiness address Province
business_address_city     VARCHAR(100)             - City/Regency Business address
business_address_subdistrict VARCHAR(100)          - District Business address
business_address_village  VARCHAR(100)             - Village Business address
business_address_detail   TEXT                     - Detail Business address
business_duration         VARCHAR(50)              - Length of business time
social_media_platform     VARCHAR(50)              - Social Media platform
social_media_account      VARCHAR(100)             - Social media account name
